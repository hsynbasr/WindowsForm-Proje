# WindowsForm-Proje
Projede, kullanıcının arayüz üzerinden bir araç ilanı verebileceği, kaydedebileceği ve silebileceği bir sistem bulunuyor. Ayrıca kullanıcılar, sisteme kaydolup oturum açarak kendi ilanlarını yönetebiliyor.

##Kullanılan Teknolojiler
* C# programlama dili 
* .NET Framework 
* Windows Forms teknolojisi 
* Microsoft SQL Server 
* Entity Framework ORM aracı 
* Visual Studio 2019
